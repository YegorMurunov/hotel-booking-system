# Hotel Booking & Pricing System

## Опис проєкту
Ця система призначена для автоматизації розрахунку вартості проживання в готелі. Проєкт розроблений на мові Java з дотриманням принципів ООП.

Система дозволяє:
- Обирати категорію номера (Standard, Deluxe, Suite, President).
- Розраховувати базову вартість на основі кількості днів.
- Автоматично застосовувати націнку за "високий сезон".
- Нараховувати гнучкі знижки для VIP-клієнтів та при тривалому проживанні (понад 14 днів).

## Технології
- **Мова програмування:** Java 11+
- **Система контролю версій:** Git

## Структура коду
- `Booking.java` — основний сервіс розрахунку ціни та валідації.
- `Room.java` — інтерфейс для об'єктів розміщення.
- `RoomType.java` — перерахування (Enum) з базовими тарифами.
- `Main.java` — демонстраційний клас із тестовими сценаріями.
